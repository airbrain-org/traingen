Detailed Description

Class added to particle systems to allow the ImageGenerator class to initiate the randomization of the particle system settings.

Member Function Documentation

◆ IsRandomizeFinished()

bool ParticleObservation.IsRandomizeFinished ( )

Called by the ImageGenerator to determine if the radomization process for this particular Observation is completed. A new image will not be generated until the randomization process is completed.

Returns
If true, the randomization process is finished and image generation may commence. If false, the radomization process is not yet complete.

◆ Randomize()

void ParticleObservation.Randomize ( )

Initiate the randomization of the appearance of the particle system for this object. This is accomplished by changing the value of certain settings using random values within a predefined range.
